# 划词翻译 Edge 插件

一个简单易用的 Edge 浏览器划词翻译插件，调用谷歌翻译 API。

## 功能特性

- ✅ 划词后显示悬浮翻译按钮
- ✅ 调用谷歌翻译 API 进行翻译
- ✅ 只显示翻译结果，不显示原文
- ✅ 翻译结果弹窗可任意拖动（边缘拖动）
- ✅ 支持选择目标语言，默认简体中文
- ✅ 支持所有网页

## 安装方法

### 1. 打开 Edge 扩展管理页面

在 Edge 浏览器地址栏输入：`edge://extensions/`

### 2. 开启开发者模式

点击页面左下角的「开发人员模式」开关，将其打开

### 3. 加载已解压的扩展程序

点击「加载解压缩的扩展」按钮，选择本插件所在的文件夹

## 使用方法

1. **设置目标语言**（可选）：点击浏览器工具栏的插件图标，选择目标语言（默认简体中文）
2. 在任意网页上，用鼠标选中（划取）想要翻译的文本
3. 鼠标松开后会在划词位置附近出现蓝色的「翻译」悬浮按钮
4. 点击「翻译」按钮，等待翻译完成
5. 翻译结果将显示在弹窗中
6. 拖动弹窗顶部边缘可移动弹窗位置
7. 点击弹窗右上角的 × 或点击页面其他位置可关闭弹窗

## 注意事项

- 需要能够访问谷歌翻译 API（translate.googleapis.com）
- 插件会在所有网页上运行
- 翻译目标语言默认为简体中文（zh-CN），可在插件设置中修改
- 更改目标语言后需要刷新页面生效
- 支持 12 种目标语言：简繁中文、英、日、韩、法、德、西、俄、意、葡、阿拉伯语
